Packaged versions of Zotero for Debian-based systems

This repository contains packaged releases of Zotero for Debian-based Linux systems and Crostini-enabled chromebooks. It updates to new releases of Zotero within 24 hours, usually faster.

The packages include the Zotero binaries as built by Zotero team themselves. They provide a system-wide installation (into the /usr/lib directory), as opposed to a single-user installation (e.g.Â in your HOME directory). They manage both desktop file registration and MimeType registration.

Installing Zotero

To install Zotero, use the following commands:

wget -qO- {site}/install.sh | sudo bash
sudo apt update
sudo apt install zotero

Note

You can use curl instead of wget by typing

curl -sL {site}/install.sh | sudo bash

Updating Zotero

Zotero as provided by this repository has its self-update facility disabled. Simply rely on on your system's package manager to give you update notifications when a new version comes out.

Alternatively, you can use the following commands:

sudo apt update
sudo apt upgrade
